Supabase 연결이 안 되어있어요. 아래 단계를 마치면 로그인 화면으로 자동 이동합니다.

1. Supabase 프로젝트 만들기
supabase.com/dashboard에서 새 프로젝트 생성 (무료 티어). 비밀번호와 리전(Tokyo 권장)을 정하고 ~2분 기다리면 준비 완료.
2. SQL 마이그레이션 4개 실행
왼쪽 메뉴 SQL Editor → tapin2/supabase/migrations/ 폴더의 4개 파일을 순서대로 붙여넣고 실행:
① 0001_schema.sql ② 0002_rls.sql ③ 0003_functions.sql ④ 0004_realtime.sql
3. Auth Hook 등록
Authentication → Hooks → "Custom Access Token" 활성화 → Function: public.set_jwt_claims 선택 후 Save
4. Email 인증 설정
Authentication → Providers → Email에서 "Confirm email"을 꺼주세요 (개발 중 권장)
5. URL과 anon key를 .env.local에 입력
Project Settings → API에서 Project URL과 anon public 키를 복사해서
tapin2/.env.local 파일을 열어 다음 두 줄에 붙여넣기:

VITE_SUPABASE_URL=https://xxxx.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGc...
6. dev 서버 재시작
터미널에서 Ctrl+C 후 다시 npm run dev
